# Proyecto Personal: Practicando HTML, CSS y JS

Este es un proyecto de carácter personal con el objetivo principal de **practicar y mejorar habilidades en HTML, CSS y JavaScript**.  

La idea del proyecto fue tomada de este video de YouTube: [Video Original](https://www.youtube.com/watch?v=drXkf_rytRs), pero ha sido adaptada y modificada a gusto propio, añadiendo cambios personales y experimentando con el diseño y la funcionalidad.

## Objetivos del Proyecto

- Practicar la estructura de HTML y semántica de etiquetas.
- Mejorar habilidades de diseño con CSS, incluyendo estilos responsivos y animaciones.
- Explorar interactividad mediante JavaScript.
- Aprender a organizar y mantener un proyecto web de forma ordenada.

## Tecnologías Utilizadas

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**
